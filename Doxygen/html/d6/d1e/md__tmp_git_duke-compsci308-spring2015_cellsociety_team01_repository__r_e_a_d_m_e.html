Use: Run Program from main. Choose simulation type with File->Load XML and selecting a .xml file from the XMLFiles folder. Run the simulation with Play or step with Step. Pause will stop the simulation, and Reset will re-initialize the grid. The slider will change how fast the program runs.

Options for changing parameters in the XML Files for testing are as follows: type: AntForaging, GameOfLife, PredatorPrey, Segregation, SlimeMolds, SpreadingOfFire, Sugarscape grid_shape: Square, Triangle grid_edge: Finite, Toroidal grid_outline: Yes, No cell_placement: Location, Percent, Random sim_color_scheme: Default, Water, Space sim_*simulation specific config variable*: any int/double grid width: any int grid height: any int team options in grid: empty, full, teamA, teamB, teamC xVals/yVals: space separated ints as locations in the grid population_percent: double between 0.0 and 1.0
