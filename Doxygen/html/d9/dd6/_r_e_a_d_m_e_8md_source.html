19 **Use:**
20 Run Program from main. Choose simulation type with File->Load XML and selecting a .xml file from the XMLFiles folder. Run the simulation with Play or step with Step. Pause will stop the simulation, and Reset will re-initialize the grid. The slider will change how fast the program runs.
